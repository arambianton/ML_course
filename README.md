# Практические работы по машинному обучению (Machine Learning)

Данный репозиторий содержит коллекцию практических и исследовательских работ по курсу **машинного обучения**.

Работы охватывают основные методы анализа данных и машинного обучения: от базовой обработки данных и визуализации до построения и оценки моделей регрессии, классификации и методов опорных векторов. Каждая работа оформлена в виде Jupyter Notebook и сопровождается экспериментами, анализом результатов и выводами.
